## 1. 게임 정체성

**한 줄 정의**
> "건설 절차가 곧 게임이다. 안전은 선택이 아니라 게임 규칙이다."

**불사(不死)의 의미**
죽지 않고 끝까지 살아남아 건물을 완성하는 것.
캐릭터가 리셋되지 않고 끝까지 키워나가는 것이 핵심 목표.

**교육 + 게임 동시 달성**
가이드대로 따라가면 완성할 수 있다.
가이드를 무시하면 사고가 난다.
교육이 곧 게임플레이.

## 2. 플레이어 역할 구조

```
신입 현장인부 (시작)
    행위 기반 작업으로 숙련도 + 돈 축적
        ↓
현장반장 (NPC 고용 가능)
    팀 편성, 작업 지시, 공정 관리
        ↓
관리자 / 발주자 (본인 돈으로 건물 올림)
    부지 매입, 설계, 발주, 수익화
        ↓
멀티 전환 시 (v3.0)
    NPC 자리를 실제 플레이어가 채움
    실제 사람 고용, 일당 지급
```

**핵심 원칙**
- 돈이 곧 자유도
- 돈 없으면 직접 다 해야 함
- 돈 생기면 시킬 수 있음
- 많이 생기면 내 건물을 짓는 목표 달성

## 3. 시점 시스템

```
기본 시점: 3인칭 (GTA 스타일)
├── WASD 이동
├── 마우스/터치 카메라 회전
└── E키 / 탭 상호작용

장비 탑승 시: 1인칭 자동 전환
├── 굴착기
├── 타워크레인
├── 펌프카
└── 기타 중장비

장비 조작 난이도: 단순하고 직관적
└── GTA처럼 쉽게, 시뮬레이터 수준 금지

핵심 원칙
└── 플레이어가 하는 모든 것을 NPC도 동일하게 수행 가능
    NPC가 굴착기 운전 → 플레이어도 굴착기 운전 가능
    NPC가 난간 설치 → 플레이어도 난간 설치 가능
```

## 4. 命 게이지 시스템

### 개인 命 게이지
```
시각화: 캐릭터 몸 안에 물이 차오르는 수위 표현
├── 위험 구역 진입 시 수위 상승
├── 안전조치 완료 구역 → 수위 하강
├── 개인보호구 착용 시 상승 속도 감소
└── 만수위 → 사고 발생 (우연 확률 적용)

위치 기반 위험 구역 예시
├── 크레인 양중 하부 → 급상승
├── 굴착 단부 (난간 미설치) → 상승
├── 전기 작업 구역 → 상승
├── 장비 작업반경 내 → 상승
└── 안전조치 완료 구역 → 유지 또는 하강

개인보호구별 효과
├── 안전모: 낙하물 위험 감소
├── 안전대: 추락 위험 감소
├── 안전화: 전기/낙하물 발 보호
├── 방진마스크: 분진 위험 감소
└── 각반: 발목 보호
```

### 현장 위험도 맵
```
구역별 색상 오버레이 (토글 가능)
├── 녹색: 안전 구역
├── 황색: 주의 구역
└── 적색: 위험 구역

안전조치 완료 시 색상 변화
└── 적 → 황 → 녹
```

### 팀 위험도 (관리자 시점)
```
팀원 개별 命 게이지 평균
└── 높으면 감독관 등장 확률 상승
```

## 5. 사고 시스템

### 하인리히 법칙 적용
```
아차사고 (300) → 경상 (29) → 중상/사망 (1)
트리거 하나 = 아차사고 확률 증가
여러 트리거 겹침 = 사고 확률 급등
재해 정도는 손실우연의 법칙으로 랜덤 결정
```

### 재해 등급별 결과
```
경상
├── 캐릭터 비틀거림 + 이펙트
├── 화면 살짝 붉어짐
├── 바로 회복
└── 특정 능력치 일시 감소

중상
├── 캐릭터 쓰러짐 + 슬로우모션 0.5초
├── 사고 원인 하이라이트 (어디서 뭘 안 했는지 표시)
├── 화면 페이드아웃 → 병원 컷씬
└── 영구 디버프 적용
    손 부상: 철근 배근 속도 -15%
    허리 부상: 자재 운반 중량 한계 -20%
    청력 손실: NPC 지시 인식률 -30%
    눈 부상: 시야 범위 영구 축소

사망
├── 즉각 연출 (GTA 사망 스타일)
├── 카메라 줌아웃 → 사고 현장 전체 보여줌
├── 사고 경위 요약 팝업
│   예: "난간 미설치 → 단부 접근 → 추락"
├── 노동부 감독관 즉시 등장
└── 캐릭터 리셋
    사라지는 것: 레벨/숙련도, 게임머니, 진행중 공사
    남는 것: 건물 소유권, 완공 이력, 캐릭터 외형 일부
```

### 노동부 감독관 등장 조건
```
├── 완전 랜덤 타이머 (백그라운드)
├── 팀 命게이지 평균 70% 이상 → 확률 급등
└── 사고 발생 → 무조건 등장

감독관 점검 항목
├── 안전보건표지 부착 여부
├── 개인보호구 착용 여부
├── 안전시설 설치 여부
└── 법적 서류 완비 여부

결과
├── 위반 항목 → 과태료
├── 사망사고 → 작업중지 + 뉴스 노출
└── 뉴스 노출 → 다음 발주 수주 확률 감소
```

## 6. 경제 시스템

### 수입
```
├── 공정 완료 보수 (발주자 지급)
├── 무사고 보너스
├── 공기 단축 보너스
├── 기능 숙련도 향상 → 시급 상승
├── 건물 임대 수익 (오프라인도 쌓임)
└── 장비 임대 수익
```

### 지출
```
├── 인건비 (일당 × 인원 × 일수)
├── 장비 임대비
├── 자재비
├── 안전시설 구매비
├── 토질조사 / 매설물 조사 비용
├── 과태료 (위반 시)
├── 피해보상금 (사고 시)
└── 공사중지 패널티 (사망사고 시)
```

### 손익
```
완공 후 순이익 = 수입 - 지출
→ 다음 프로젝트 자본금으로 이월
```

## 7. NPC 시스템

### NPC 등급
```
숙련공 (10년차)
├── 정확도 85%
├── 자율판단 가능
└── 피로 누적 시 실수 확률 상승

반숙련공 (3년차)
├── 정확도 65%
├── 감독 필요
└── 지시 없으면 안전조치 스킵 확률 높음

신입 (1년차)
├── 정확도 40%
├── TBM 필수
└── 시범 안 보이면 엉뚱하게 작업

외국인 (아흐마드/응우옌)
├── 한국어 이해도 60%
├── 언어 오해 확률 40%
└── 그림/몸짓 TBM 필요
```

### NPC 실수 확률 계산
```javascript
mistakeRate = (fatigue/100 × 0.3) + ((1 - skill) × 0.2) + languageGap × 0.4
// TBM 실시 시: mistakeRate × 0.6
// 시범 보임 시: mistakeRate × 0.5
// 그림TBM 시: languageGap × 0.3
```

### NPC 상태 변수
```
├── 숙련도 (작업 정확도)
├── 피로도 (시간 누적, 야간작업 시 급증)
├── 언어 이해도
├── 안전 인식도 (TBM 실시 시 상승)
└── 건강 상태 (부상 시 능력치 저하)
```

### NPC 사망
```
영구 소실 — 새로 고용 필요
→ 관리자로서의 책임감 부여
→ NPC 생명도 소중하게 관리하게 되는 구조
```

## 8. 성장 시스템

### 행위 기반 숙련도
```
모든 행위 → 해당 직종 경험치 적립
├── 굴착기 조작 100회 → 굴착기 운전원 숙련도 상승
├── 철근 배근 200회 → 철근공 숙련도 상승
└── 모든 공종 골고루 → 올라운더 경로
```

### 숙련도 단계
```
Lv1 잡부      모든 작업 가능, 속도 느림, 실수율 높음
Lv2 보조공    특정 공종 속도 향상
Lv3 기능공    실수율 감소, 일당 상승
Lv4 숙련공    NPC 지도 가능, 작업 품질 상승
Lv5 전문공    해당 공종 최고 효율
Lv6 반장      팀 편성 권한, 관리 가능
```

### 멀티 직종 성장
```
철근공 Lv5 + 형틀공 Lv3 + 굴착 Lv2
→ 올라운더는 오픈월드에서 몸값 높음
→ 어디서든 불려다니는 구조
```

## 9. 튜토리얼 공정 흐름

### 목표: 1시간 완주, 5층 건물 완성

### 시작 세팅 (현장사무소)
```
부지 도면 확인
├── 면적 확인 → 사면개착 가능 여부 판단
├── 인접 건물/도로 확인
└── 지형 경사 확인

사전 조사 발주
├── 토질조사 업체 선택 (비용/정확도 차이)
└── 지하매설물 조사 (가스/전기/상하수도)

굴착 공법 선택
├── 사면개착 (여유 부지 + 양호한 토질)
│   토질별 경사도 자동 계산
│   모래 1:1.8 / 점토 1:1.2 / 암반 1:0.5
└── 흙막이 공법
    엄지말뚝+토류판 / CIP / SCW

장비 & 인력 편성
├── 굴착기 (직접 운전 or NPC 기사 고용)
├── 덤프트럭 대수 결정
└── 초기 인력 편성 (보유 자금 기준)

안전 계획
├── 보호구 구매
├── 안전시설 구매
└── 착공 전 서류 준비
```

### 공정별 체크리스트 (예시: 토공사)
```
필수 안전조치
☐ 지하매설물 조사 완료
☐ 토질조사 결과 확인
☐ 굴착 단부 난간 설치
☐ 작업반경 라바콘 설치
☐ 사면 경사도 확인

작업 순서
☐ 착공신고 완료
☐ TBM 실시
☐ 굴착기 배치
☐ 굴착 진행
☐ 흙막이 설치 (해당 시)
☐ 집수정 위치 지정
☐ 버팀보 설치 (해당 시)
```

### 튜토리얼 타임라인
```
00:00 캐릭터 생성 + 현장 도착
00:05 준비단계 (토질조사, 매설물 조사, 공법 선택)
00:15 토공사 (굴착, 흙막이, 난간, 집수정)
00:30 기초공사 (먹메김, 버림타설, 철근배근, 기초타설)
00:45 지상공사 (기둥/보/슬라브 1~5층)
00:55 마무리 (무사고 완성 시 수료증 발급)
01:00 END
```

## 10. 힌트 & 가이드 시스템

### 선배 NPC
```
튜토리얼 전용 선배 NPC 항상 동행
├── "여기서는 이걸 먼저 해야 해요"
├── 실수 시 "괜찮아요, 이렇게 하면 돼요"
├── 위험 근접 시 "위험해요! 물러서세요"

### 힌트 강도
```
초보: 자동 경고 + 방법까지 안내 (기본값)
일반: 경고만
하드: 없음 (추후 추가)
```

### 안전조치 미이행 시
```
체크리스트 미완료 상태로 다음 공정 진행 시도
→ "미완료 항목이 있습니다. 계속하시겠습니까?"
→ 강행 시 해당 항목 위험도 누적 시작
```

## 12. 아트 디렉션

### 캐릭터
```
비율: SD 2~2.5등신
├── 머리 : 몸 = 1 : 1.5~2
├── 머리 크고 둥글게
├── 눈 크고 단순하게 (점 or 동그라미)
├── 팔다리 짧고 통통하게
└── 손발 뭉툭하게

레퍼런스: 동물의 숲보다 살짝 길쭉
         곰돌이 + SD 드래곤 중간 비율
         동물이 아닌 사람, 3D

개인보호구 착용 시각화
├── 안전모 (색상, 스티커 커스텀)
├── 안전대 (착용 여부 시각화)
├── 안전화 / 각반
├── 장갑 / 조끼
└── 직종별 작업복
→ 보호구가 캐릭터보다 상대적으로 크게 보이는 효과
→ 귀여움 + 교육적 직관성 동시 달성
```

### 현장 & 건물
```
톤: 맑고 밝음 + 현실적 (중간)
날씨에 따라 분위기 전환

건물/차량/장비: 사실적 (실제 치수 기반)
├── 콘크리트 타설면: PBR 텍스처 (거친 표면, 타설 자국)

## 20. 오픈월드 구조 (v2.0 이후)

### 맵 구조
```
하나의 큰 맵
├── 초기 상태: 빈 부지들만 있는 황량한 맵
├── 플레이어들이 건물 올리면서 맵이 채워짐
├── 구역별 용도지역 (주거/상업/공업/녹지)
└── 한국/일본/유럽 각 구역 존재
```

### 도시 발전 단계
```
Phase 1 황무지 → Phase 2 공사판
→ Phase 3 신도시 초기 → Phase 4 생활권
→ Phase 5 도시
```

### 건물 수익 구조
```
주거: 임대 수익 (오프라인도 쌓임)
상업: 방문객 수 기반 수익
공업: 생산/물류 수익
복합: 수익 다각화
```

### 실패 동기부여
```
리셋되더라도 남는 것
├── 건물 소유권 (내가 지은 건물은 안 없어짐)
├── 완공 이력 (서버에 영구 기록)
└── 캐릭터 외형 아이템 일부
→ "내 건물은 남는다" = 다시 시작하고 싶게 만드는 핵심
```

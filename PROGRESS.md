# BULSA 개발 현황
> 작업 완료 시 이 파일만 업데이트. 내용은 항상 30줄 이내로 유지.

## 마지막 완료 작업
- **수직 충돌 버그 픽스 (벽 텔레포트)**: `maxStepUp = 0.45m` 상한 — 벽 윗면 잡혀 텔레포트되던 문제 차단
- **양중 빔 흔들림 (cannon.js 펜듈럼 효과)**: 상승 중 ±6cm sin/cos 진동 + 살짝 기울임, 거치 시 안정
- **안전관리자 적발 시스템 (`core/inspector.js`)**:
  · 40~80초 후 사무실 쪽에서 inspector NPC 출현 → 플레이어 접근 → 다이얼로그
  · 5개 시나리오별 `getViolations_*()` — 산안법 시행령 별표 35 기준 과태료 산정
  · 누적 과태료 HUD (`💰 ₩…`) + 완주 패널 표시
  · `INTERACTION.popupOpen` / `gameOver` / `liftStarted` 동안 미출현
- **물리 데모 + 5시나리오 자동 테스트 인프라** (이전 커밋)

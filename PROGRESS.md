# BULSA 개발 현황
> 작업 완료 시 이 파일만 업데이트. 내용은 항상 30줄 이내로 유지.

## 마지막 완료 작업
- 평면도 레이아웃 개선: 건축 도면 스타일 (이중벽·해치·구조그리드·범례·치수선·북향·축척바)

## 다음 작업 (우선순위 순)
1. **Phase 3: equipment_setup** — 크레인 아웃트리거 펼치기, 수준기 확인, 과부하방지장치 점검 UI
2. **크레인 운전원 NPC 거부권** — 안전기준 미충족 시 크레인 운전원이 작업 거부하는 로직
3. **Phase 1·2 연동 스텁** — 무사이 계획서 완료 후 BULSA 진입 트리거 (기본값으로 자동 통과)

## 핵심 결정사항
- Phase 1(작업계획서)·Phase 2(안전검토)는 무사이 담당. BULSA v1.0은 Phase 3부터 시작
- 추가 시나리오(밀폐/전기/화재/비계/차량)는 v2.0 스텁으로 유지, 현재 삭제 불가
- Rapier.js 물리엔진은 v2.0으로 미룸

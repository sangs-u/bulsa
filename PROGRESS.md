# BULSA 개발 현황
> 작업 완료 시 이 파일만 업데이트. 내용은 항상 30줄 이내로 유지.

## 마지막 완료 작업
- 작업반경 진입 감지: Phase 6 이후 또는 인양 진행 중 반경(7.7m) 내 진입 시 토스트 경고·화면 비네트·안전지수 -2·위반 카운터(HUD), 4개국어

## 다음 작업 (우선순위 순)
1. **운전원 NPC 시각화** — 현재는 팝업만, 실제 NPC 메시(헬멧·조끼)를 운전석 옆에 배치
2. **미보호장구 감지** — 안전모·안전화 등 PPE 미착용 상태 검출 (인벤토리 시스템 필요)
3. **NPC 자동 대피** — 인양 시작 시 작업반경 내 NPC 들이 자동 이탈하는 AI 로직
4. **무사이 연동 스텁** — 보류 (게임 콘텐츠 우선 개발)

## 핵심 결정사항
- Phase 1(작업계획서)·Phase 2(안전검토)는 무사이 담당. BULSA v1.0은 Phase 3부터 시작
- 추가 시나리오(밀폐/전기/화재/비계/차량)는 v2.0 스텁으로 유지, 현재 삭제 불가
- Rapier.js 물리엔진은 v2.0으로 미룸

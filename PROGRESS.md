# BULSA 개발 현황
> 작업 완료 시 이 파일만 업데이트. 내용은 항상 30줄 이내로 유지.

## 마지막 완료 작업
- 토공사 6단계 흐름 완성: 작업계획서 UI(깊이·구배·흙막이·매설물 입력) → 매설물 조사 → 흙막이 → 안전난간 → 신호수 → 굴착기 운전석·실행. EXCAV_STATE + evaluateExcavation + 4개 사고 유형 연결. HUD·accident 시스템 시나리오 인식 분리

## 다음 작업 (우선순위 순)
1. **토공사 4국어 i18n + 사이클별 차등화** — 현재 한글 위주, 영문/베트남/아랍어 + 작업계획서 hint 변주
2. **과태료/감독관 이벤트 시스템** — applyPenalty + 감독관 NPC + 위반 검증
3. **공정 간 자동 전환** — 토공사 완료 → 기초공사 자동 진입 (또는 ?s=foundation 라우팅)
4. **기초공사 모듈 신규** — manifest의 두 번째 공정 골조 시작

## 핵심 결정사항
- Phase 1(작업계획서)·Phase 2(안전검토)는 무사이 담당. BULSA v1.0은 Phase 3부터 시작
- 추가 시나리오(밀폐/전기/화재/비계/차량)는 v2.0 스텁으로 유지, 현재 삭제 불가
- Rapier.js 물리엔진은 v2.0으로 미룸
